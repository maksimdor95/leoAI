# Husky Pre-commit Hooks

Эта папка содержит настройки для автоматической проверки кода перед коммитом.

## Установка

После установки зависимостей проекта (`npm install`), выполните:

```bash
npx husky install
```

Хуки будут автоматически запускаться при каждом `git commit`.

## Что проверяется

- ESLint - проверка JavaScript/TypeScript кода
- Prettier - форматирование кода
- Black - форматирование Python кода (если установлен)
- isort - сортировка импортов в Python (если установлен)

## Отключение для одного коммита

Если нужно пропустить проверку:

```bash
git commit --no-verify -m "Your message"
```
