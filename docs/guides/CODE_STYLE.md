# Стиль кода

## JavaScript / TypeScript

### ESLint

```bash
npm run lint          # Проверить код
npm run lint:fix      # Исправить автоматически
```

### Prettier

```bash
npm run format        # Отформатировать код
npm run format:check  # Проверить форматирование
```

### Правила

- Одинарные кавычки (`'`)
- Точка с запятой обязательна
- Ширина строки: 100 символов
- Отступ: 2 пробела

## EditorConfig

Настройки из `.editorconfig` автоматически применяются в поддерживаемых редакторах: кодировка, отступы, переводы строк.
