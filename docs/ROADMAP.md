# AIheroes Platform Roadmap

## Обзор

Дорожная карта развития платформы AIheroes — мульти-продуктовой системы AI-ассистентов для карьерного развития.

---

## Выполнено (MVP)

### Базовая платформа

- [x] **Microservices архитектура** — Conversation, AI/NLP, User Profile, Job Matching, Email сервисы
- [x] **Авторизация** — JWT токены, User Profile Service
- [x] **Redis** — хранение сессий и состояния диалогов
- [x] **PostgreSQL** — хранение пользователей и вакансий
- [x] **REST API** — совместимость с Yandex Serverless Containers
- [x] **WebSocket** — для локальной разработки

### Jack (LEO) — AI-ассистент для поиска работы

- [x] **Сценарий диалога** (`jack-profile-v2`) — 20+ шагов
- [x] **Сбор профиля кандидата** — опыт, навыки, предпочтения
- [x] **Job Matching** — подбор вакансий по профилю
- [x] **Email уведомления** — персонализированные письма с вакансиями
- [x] **Голосовой ввод** — Web Speech API (STT)
- [x] **Озвучка вопросов** — Web Speech API (TTS)

### wannanew — подготовка PM к собеседованиям

- [x] **Сценарий диалога** (`wannanew-pm-v1`) — 10 шагов
- [x] **Сбор профиля PM** — уровень, тип продукта, опыт
- [x] **Голосовое интервью** — 3 типовых PM-вопроса
- [x] **Метки продукта** — различение в списке чатов

### Frontend

- [x] **Landing page** — регистрация, вход
- [x] **Chat interface** — диалог с AI
- [x] **Chats list** — список чатов с превью
- [x] **Profile modal** — просмотр собранного профиля
- [x] **Темная тема** — современный дизайн
