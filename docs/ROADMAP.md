# План развития (Roadmap)

## 1. Текущий статус (MVP 0)

**Цель:** Стабильный end-to-end путь для Jack и WannaNew.
- [x] Инфраструктура (Docker, VPS, HTTPS).
- [x] Базовые сценарии (Чат, Вакансии, Отчёты).
- [x] Авторизация и OAuth.
- [x] Smoke-тесты (daily gate).

---

## 2. Ближайшие задачи (High Priority)

1. **OAuth Production:** Настройка реальных credentials для Google/Yandex.
2. **Voice Hardening:** Переход на серверный SpeechKit для стабильности.
3. **E2E Testing:** Полное покрытие путей регистрации и завершения сценариев.
4. **Monitoring:** Подключение Sentry и базовых алертов.

---

## 3. План MVP 1 (Event-Driven & Scale)

### Эпики
- **Epic A: Event Backbone.** Перевод тяжелых задач (Email, Report) на асинхронную модель (BullMQ/Redis).
- **Epic B: AI Quality.** Внедрение системы оценки качества ответов (Eval pipeline).
- **Epic C: Search Upgrade.** Векторный поиск (Qdrant) и улучшение ранжирования вакансий.
- **Epic D: Reliability.** Сквозное логирование (Correlation ID) и SLO-дашборды.

### Таймлайн
- Недели 1-4: Инфраструктура событий, асинхронные воркеры.
- Недели 5-8: Улучшение поиска и качества AI.
- Недели 9-10: Стабилизация и релиз MVP 1.

---

## 4. Долгосрочный Backlog

- **Resume Agent:** Автоматическое улучшение и генерация резюме.
- **Skills Graph:** Глубокий анализ навыков и построение траектории обучения.
- **B2B Service (Jill):** Инструменты для рекрутеров по поиску кандидатов.
- **Mobile App:** Мобильная версия на React Native.
