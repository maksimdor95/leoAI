# Стратегия роста и масштабирования Jack & Jill AI

## Исполнительное резюме

**Цель:** Достижение product-market fit на рынке России/СНГ за 6-12 месяцев и привлечение раунда инвестиций для масштабирования с последующей стратегической продажей.

**Текущее состояние:** MVP с core функциональностью (Conversation Service, AI/NLP, Frontend). Отсутствуют критичные сервисы: Job Matching, Email Notification, Referral Service.

### Ключевые метрики для достижения

- 10,000+ зарегистрированных пользователей
- 5,000+ активных диалогов в месяц
- 50+ компаний-партнеров (Jill side)
- MRR $50K+ (от компаний)
- NPS > 50
- Conversion rate: диалог → успешное размещение > 15%

---

## Фаза 1: Доведение MVP до Production-Ready (0-3 месяца)

### Месяц 1: Критичная функциональность

**Цель:** Реализовать полный цикл value delivery для кандидатов

#### Приоритетные задачи

**1. Job Matching Service (MVP)**

- Интеграция с российскими джоб-бордами: HH.ru API, Avito Jobs, Работа.ру
- Базовый rule-based matching алгоритм
- Schema для вакансий в PostgreSQL
- Endpoint: `GET /api/jobs/match/:userId`
- **Критерий успеха:** 100+ вакансий в базе, matching работает

**2. Email Notification Service (MVP)**

- Интеграция с SendGrid/Yandex Mail
- Базовые email templates на русском
- Персонализированные подборки вакансий
- Endpoint: `POST /api/email/send-jobs`
- **Критерий успеха:** Отправка 100+ email в неделю

**3. Production Infrastructure**

- Deploy на Yandex Cloud или Selectel
- Настройка CI/CD pipeline
- Мониторинг базовый (Prometheus + Grafana)
- Backups баз данных
- **Критерий успеха:** Uptime > 99%, деплой автоматизирован

#### Ресурсы

- **Команда:** 2-3 backend разработчика, 1 DevOps (part-time)
- **Бюджет:** $5,000-8,000 (инфраструктура + разработка)

---

### Месяц 2: Запуск закрытой беты

**Цель:** Привлечь 100-200 первых пользователей и собрать feedback

#### Приоритетные задачи

**1. Маркетинг и привлечение пользователей**

- Landing page оптимизация для русскоязычной аудитории
- Запуск в закрытую бета-группу (Telegram, VK, LinkedIn)
- Партнерство с карьерными сообществами (Career.ru, Habr Career)
- Контент-маркетинг: статьи про AI в рекрутинге

**2. Improvements на основе feedback**

- Улучшение промптов Jack под российский рынок
- Оптимизация диалоговых сценариев
- Исправление критичных багов

**3. Analytics и метрики**

- Внедрение аналитики (Amplitude/Mixpanel)
- Dashboard для ключевых метрик
- Сбор данных для улучшения matching

#### Критерии успеха

- 100+ зарегистрированных пользователей
- 50+ завершенных диалогов
- NPS > 40
- 10+ положительных отзывов

**Бюджет:** $2,000-3,000 (маркетинг, инструменты аналитики)

---

### Месяц 3: Оптимизация и масштабирование

**Цель:** Улучшить качество продукта и подготовить к публичному запуску

#### Приоритетные задачи

**1. Векторный поиск (Vector DB)**

- Интеграция с Qdrant (российский векторный DB)
- Embeddings для вакансий и профилей
- Semantic matching добавлен к rule-based
- **Критерий успеха:** Улучшение релевантности matching на 30%+

**2. Расширение источников вакансий**

- Добавление 3-5 новых источников
- Парсинг сайтов компаний
- Дедупликация вакансий
- **Критерий успеха:** 5,000+ вакансий в базе

**3. Performance optimization**

- Кеширование результатов matching
- Оптимизация запросов к БД
- Load testing и оптимизация
- **Критерий успеха:** Response time < 200ms (p95)

#### Критерии успеха Фазы 1

- 500+ пользователей
- 200+ завершенных диалогов
- 1,000+ отправленных email с вакансиями
- Matching accuracy > 70%

## Стратегия выхода на рынок

### Таргетинг

**Кандидаты (B2C):**

- IT-специалисты (разработчики, дизайнеры, PM)
- Менеджеры среднего звена
- Возраст: 25-45 лет
- География: Москва, СПб, города-миллионники

**Компании (B2B):**

- Технологические стартапы (50-200 человек)
- Средний бизнес (IT-отделы)
- Рекрутинговые агентства
- География: Москва, СПб, регионы с IT-кластерами

### Каналы привлечения

**B2C (кандидаты):**

- Контент-маркетинг: Habr, VC.ru, YouTube
- Социальные сети: VK, Telegram каналы, LinkedIn
- Партнерства: карьерные сообщества, буткемпы
- SEO: статьи про поиск работы, карьерные советы
- Referral program

**B2B (компании):**

- Outbound sales: холодные звонки, email
- Партнерства: HR-конференции, рекрутинговые агентства
- Inbound marketing: case studies, вебинары
- Account-based marketing для крупных клиентов

### Ценообразование

**B2C (Freemium):**

- **Free:** базовый доступ, 5 вакансий в неделю
- **Premium ($9.99/месяц):** неограниченные вакансии, приоритет в matching, карьерный коучинг

**B2B (Subscription + Success Fee):**

- **Starter ($500/месяц):** доступ к 100 кандидатам, базовый matching
- **Professional ($1,500/месяц):** неограниченный доступ, priority matching, analytics
- **Enterprise ($5,000+/месяц):** кастомные интеграции, dedicated support
- **Success fee:** 10-15% от годовой зарплаты при успешном найме

3. **Качество данных:** Мультиагентная система валидации и анализа
4. **Персонализация:** Уникальный опыт для каждого пользователя
5. **Техническое превосходство:** Современный стек, масштабируемая архитектура

## Риски и митигация

| Риск                                             | Митигация                                                          |
| ------------------------------------------------ | ------------------------------------------------------------------ |
| Конкуренция с HH.ru, Avito Jobs                  | Фокус на AI и персонализацию, позиционирование как премиум-решение |
| Регуляторные риски (152-ФЗ, персональные данные) | Compliance с самого начала, юридическая поддержка                  |
| Сложность масштабирования matching алгоритма     | Итеративная разработка, A/B тестирование, user feedback            |
| Недостаток финансирования                        | Bootstrap до первых метрик, затем привлечение инвесторов           |
| Отсутствие критической массы пользователей       | Фокус на узкую нишу (IT) сначала, затем расширение                 |
