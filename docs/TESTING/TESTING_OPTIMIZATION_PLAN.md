# План оптимизации тестирования

## Стратегия оптимизации

**Цель:** Минимизировать количество полных прогонов диалога для экономии API вызовов.

**Принципы:**

1. Объединять тесты в один прогон, где это возможно
2. Использовать короткие ответы для быстрого прохождения
3. Проверять несколько функций в одном диалоге
4. Минимизировать шаги до нужного теста

---

## Группировка тестов

### Группа 1: Profile Analyst (Тесты 11-13)

**Прогон 1.1: Тесты 11-12 (объединены)**

- Пропустить критичное поле (должность: "нерелевантно")
- Дойти до `additional` → проверить `completion_gap`
- На `completion_gap` ответить "заполнить" → проверить возврат
- Заполнить должность → проверить завершение

**Прогон 1.2: Тест 13 (короткий)**

- Пропустить критичное поле (должность: "нерелевантно")
- Дойти до `additional` → `completion_gap`
- Ответить "продолжить как есть" → проверить завершение

---

### Группа 2: Context Manager + Интеграция (Тесты 14-16)

**Прогон 2: Все три теста в одном диалоге**

1. Приветствие: "да"
2. Должность: нерелевантно "привет" → Validator → clarify → "Product Manager"
3. Опыт: отклонение "А что вы предлагаете?" → Context Manager → следующий вопрос с подсказкой → "5 лет, кстати ищу удаленную работу" → Context Manager (важная информация)
4. Продолжить диалог, пропустив локацию
5. Дойти до `additional` → Profile Analyst → `completion_gap`

**Проверяем:**

- ✅ Тест 14: Context Manager - отклонение от темы
- ✅ Тест 15: Context Manager - важная информация
- ✅ Тест 16: Интеграция всех агентов

---

### Группа 3: Новые функции (Тесты 17-19)

**Прогон 3: Все три теста в одном полном диалоге**

1. Пройти диалог до `tasks` → ответить → проверить `profile_snapshot` (Тест 17)
2. Нажать "Продолжить" → продолжить диалог
3. На любом шаге написать "покажи мой профиль" → проверить команду (Тест 18)
4. Продолжить диалог до `additional` → ответить
5. После завершения написать "Как составить резюме?" → проверить свободное общение (Тест 19)

**Проверяем:**

- ✅ Тест 17: profile_snapshot с кнопкой
- ✅ Тест 18: Команда "покажи мой профиль"
- ✅ Тест 19: Свободное общение
