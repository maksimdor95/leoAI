# Шаг 0.2: Настройка базовой инфраструктуры - Простой план

## 🎯 Что мы делаем в Шаге 0.2?

Шаг 0.2 - это **подготовка "инфраструктуры"** - всех систем, которые будут работать "под капотом" нашего приложения. Это как подключить коммуникации к дому: электричество, воду, интернет.

## ⏱️ Сколько времени это займет?

**Общее время: 2-3 дня** (если работаем по 4-6 часов в день)

---

## 📦 Что мы будем настраивать

### 1. Docker и Docker Compose

- **Что это:** Система "контейнеров" для запуска всех частей приложения
- **Зачем:** Чтобы всё работало одинаково на любом компьютере
- **Аналогия:** Как коробки с готовыми наборами, которые можно запустить где угодно

### 2. PostgreSQL (база данных)

- **Что это:** Большая "таблица Excel" в облаке, где хранятся данные
- **Зачем:** Хранить профили пользователей, вакансии, историю диалогов
- **Аналогия:** Как огромная библиотека, где всё организовано

### 3. Redis (кеш)

- **Что это:** Быстрая "память" для временных данных
- **Зачем:** Хранить активные сессии, временные данные диалога
- **Аналогия:** Как быстрая записная книжка под рукой

### 4. Dockerfile'ы

- **Что это:** Инструкции, как собрать каждую часть приложения
- **Зачем:** Чтобы каждый сервис мог запуститься в контейнере
- **Аналогия:** Как рецепт для каждого блюда

---

## 📋 План действий

### Действие 1: Настроить Docker Compose

**Что делаем:**

- Создаем файл `docker-compose.yml`
- Настраиваем PostgreSQL
- Настраиваем Redis
- Настраиваем сеть между сервисами

**Время:** 30-40 минут

---

### Действие 2: Создать базовые Dockerfile'ы

**Что делаем:**

- Создаем Dockerfile для каждого сервиса
- Настраиваем базовые образы
- Пока пустые (будут заполняться позже)

**Время:** 20-30 минут

---

### Действие 3: Настроить подключение к базе данных

**Что делаем:**

- Создаем скрипты для инициализации базы данных
- Настраиваем переменные окружения
- Тестируем подключение

**Время:** 20-30 минут

---

### Действие 4: Создать инструкции по запуску

**Что делаем:**

- Обновляем документацию
- Добавляем команды для запуска
- Добавляем инструкции по решению проблем

**Время:** 15-20 минут

---

## 🎯 Итоги Шага 0.2

После выполнения всех действий у нас будет:

- ✅ Docker Compose настроен
- ✅ PostgreSQL и Redis запускаются одной командой
- ✅ Базовые Dockerfile'ы для всех сервисов
- ✅ Готовая инфраструктура для разработки
