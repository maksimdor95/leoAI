# Шаг 1.2: Frontend - Базовая регистрация и авторизация

## 🎯 Что мы делаем?

Создаем веб-интерфейс (то, что видит пользователь):

- Страница регистрации
- Страница авторизации
- Красивый дизайн с Ant Design
- Интеграция с User Profile Service API

**Технологии:** React/Next.js + Ant Design + TypeScript  
**Время:** 3-4 дня

---

## 📋 План действий (по порядку)

### Действие 1: Настроить React/Next.js проект

**Что делаем:**

- Создать структуру frontend проекта
- Настроить Next.js
- Настроить TypeScript
- Установить зависимости

**Время:** 30-40 минут

---

### Действие 2: Установить и настроить Ant Design

**Что делаем:**

- Установить Ant Design
- Настроить базовую тему
- Создать layout компонент

**Время:** 20-30 минут

---

### Действие 3: Создать страницу регистрации

**Что делаем:**

- Создать форму регистрации с Ant Design
- Добавить валидацию
- Интегрировать с API (User Profile Service)
- Обработка ошибок

**Время:** 1-2 часа

---

### Действие 4: Создать страницу авторизации

**Что делаем:**

- Создать форму авторизации с Ant Design
- Интегрировать с API
- Сохранение токена
- Редирект после успешного входа

**Время:** 1-2 часа

---

### Действие 5: Настроить роутинг и навигацию

**Что делаем:**

- Настроить маршруты (регистрация, авторизация)
- Создать навигацию между страницами
- Защита страниц (требует авторизацию)

**Время:** 30-40 минут

---

### Действие 6: Создать страницу профиля

**Что делаем:**

- Создать страницу с профилем пользователя
- Отображение данных пользователя
- Возможность редактирования

**Время:** 1 час

---

## 🎯 Итог Шага 1.2

После выполнения всех действий у нас будет:

- ✅ Красивый веб-интерфейс
- ✅ Страница регистрации
- ✅ Страница авторизации
- ✅ Страница профиля
- ✅ Интеграция с User Profile Service
- ✅ Сохранение токена и авторизация
