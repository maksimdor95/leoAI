## 📚 Рекомендуемые библиотеки

### 1. Ant Design (Antd) - РЕКОМЕНДУЮ

**Сайт:** https://ant.design/components/overview/

**Что это:**

- Полнофункциональная библиотека компонентов для React
- Очень популярная (используется в Alibaba, Tencent и др.)
- Отличная документация
- Много готовых компонентов

**Плюсы:**

- ✅ Огромное количество компонентов (60+)
- ✅ Красивый дизайн из коробки
- ✅ Отличная документация
- ✅ TypeScript поддержка
- ✅ Много примеров
- ✅ Активно поддерживается

**Минусы:**

- ❌ Размер bundle может быть большим (но можно tree-shake)
- ❌ Стиль может быть слишком "корпоративным"

**Подходит для:**

- ✅ Формы (регистрация, авторизация)
- ✅ Таблицы
- ✅ Модальные окна
- ✅ Навигация
- ✅ Все типы UI компонентов

**Установка:**

```bash
npm install antd
```

**Пример использования:**

```tsx
import { Button, Input, Form } from 'antd';

function RegisterForm() {
  return (
    <Form>
      <Form.Item label="Email">
        <Input type="email" />
      </Form.Item>
      <Form.Item label="Password">
        <Input.Password />
      </Form.Item>
      <Button type="primary">Register</Button>
    </Form>
  );
}
```

## 🎯 Как использовать Ant Design в нашем проекте

### Шаг 1: Установить

```bash
cd frontend
npm install antd
```

### Шаг 2: Использовать в компонентах

```tsx
import { Button, Form, Input, Card } from 'antd';
import 'antd/dist/reset.css'; // Или antd/dist/antd.css

function RegisterPage() {
  return (
    <Card title="Регистрация">
      <Form>
        <Form.Item label="Email" name="email">
          <Input type="email" />
        </Form.Item>
        <Form.Item label="Пароль" name="password">
          <Input.Password />
        </Form.Item>
        <Button type="primary" htmlType="submit">
          Зарегистрироваться
        </Button>
      </Form>
    </Card>
  );
}
```
