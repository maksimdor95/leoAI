# Пошаговая инструкция по работе с pgAdmin

## 🎯 Что вы видите сейчас

В pgAdmin вы видите:

- **Слева:** Object Explorer (обозреватель объектов)
- **Справа:** SQL редактор с кодом создания базы данных

**База данных `jack_ai` уже создана и видна!** ✅

---

## 📋 Что делать дальше (по шагам)

### Шаг 1: Развернуть схему "jack"

**В левой панели (Object Explorer):**

1. Найдите базу данных `jack_ai` (она выделена синим)
2. Кликните на стрелочку слева от `jack_ai` (чтобы развернуть)
3. Найдите папку **"Schemas"** (Схемы)
4. Кликните на стрелочку слева от "Schemas"
5. Найдите схему **"jack"**
6. Кликните на стрелочку слева от "jack"

**Что вы увидите:**

- Разные типы объектов (Tables, Views, Functions и т.д.)

---

### Шаг 2: Найти таблицу "users"

**В левой панели:**

1. В схеме "jack" найдите папку **"Tables"** (Таблицы)
2. Кликните на стрелочку слева от "Tables"
3. Вы увидите таблицу **"users"** ✅

**Это наша таблица с пользователями!**

---

### Шаг 3: Посмотреть данные в таблице "users"

**Способ 1: Через контекстное меню (проще)**

1. **Правый клик** на таблице "users"
2. Выберите **"View/Edit Data"** (Просмотр/Редактирование данных)
3. Выберите **"All Rows"** (Все строки)

**Что вы увидите:**

- Таблица со всеми пользователями
- Колонки: id, email, password_hash, first_name, last_name, created_at, updated_at
- Данные пользователей, которых вы зарегистрировали

---

**Способ 2: Через SQL запрос**

1. **Правый клик** на таблице "users"
2. Выберите **"Query Tool"** (Инструмент запросов)
3. В открывшемся окне введите:
   ```sql
   SELECT * FROM jack.users;
   ```
4. Нажмите кнопку **▶ (Execute)** или **F5**

**Что вы увидите:**

- Результат запроса в виде таблицы
- Все пользователи из базы данных

---

### Шаг 4: Посмотреть структуру таблицы

**Как посмотреть, какие поля есть в таблице:**

1. **Правый клик** на таблице "users"
2. Выберите **"Properties"** (Свойства)
3. Перейдите на вкладку **"Columns"** (Колонки)

**Что вы увидите:**

- Список всех полей (id, email, password_hash и т.д.)
- Типы данных каждого поля
- Ограничения (NOT NULL, UNIQUE и т.д.)

---

## 🔍 Полезные SQL запросы

### Посмотреть всех пользователей:

```sql
SELECT * FROM jack.users;
```

### Посмотреть только email и имена:

```sql
SELECT email, first_name, last_name FROM jack.users;
```

### Посчитать количество пользователей:

```sql
SELECT COUNT(*) FROM jack.users;
```

### Посмотреть последнего зарегистрированного:

```sql
SELECT * FROM jack.users ORDER BY created_at DESC LIMIT 1;
```

### Найти пользователя по email:

```sql
SELECT * FROM jack.users WHERE email = 'oxan@example.com';
```

## 🎯 Быстрая справка

### Как открыть Query Tool (SQL редактор):

1. Правый клик на объекте (база, таблица, схема)
2. Выберите "Query Tool"

### Как выполнить SQL запрос:

1. Введите SQL код
2. Нажмите **▶ (Execute)** или **F5**

### Как посмотреть данные таблицы:

1. Правый клик на таблице
2. "View/Edit Data" → "All Rows"

### Как выйти из pgAdmin:

- Просто закройте вкладку браузера
