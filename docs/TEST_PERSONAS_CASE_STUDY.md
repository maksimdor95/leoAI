# Кейс-тест: Персоны для проверки продукта LEO AI

## Цель тестирования

Проверить профпригодность продукта через три реалистичных пользовательских сценария:
- Корректность фильтрации вакансий (role-family, scoring)
- Качество и универсальность промптов (не-IT роли, нестандартные запросы)
- Работоспособность интеграции с YandexGPT
- Безопасность (JWT, данные пользователя)
- Удобство UX (retention signal: решил ли проблему, вернётся ли)

## Сценарий тестирования (для каждой персоны)

### Фаза 1: Онбординг
1. Регистрация / логин
2. Выбор сценария «Подбор вакансий» (Jack)
3. Оценка первого приветственного сообщения

### Фаза 2: Диалог (сбор профиля)
4. Quick path: минимум вопросов → первые вакансии
5. Detailed path: полный сбор → качество вакансий
6. Тест на off-topic (проверка context manager)
7. Тест на неясный ответ (проверка validator)

### Фаза 3: Результат
8. Список вакансий — релевантность (score, family)
9. Отсутствие «мусорных» вакансий (cross-family noise)
10. Catalog warning при малом покрытии (Светлана)
11. Email-дайджест — доставка и содержание

### Фаза 4: Оценка
12. Решена ли проблема пользователя? (1-5)
13. Вернётся ли пользователь? (да/нет + почему)
14. Рекомендация по улучшению

## Стресс-тесты (провокации для проверки системы)

### Off-topic тесты (context manager)
- Иван: «Кстати, а вы можете помочь мне написать SQL-запрос для текущей работы?»
- Андрей: «А можно мне просто шаблон коммерческого предложения?»
- Светлана: «Мне тревожно из-за поиска работы, можете меня проконсультировать?»

### Неясные ответы (validator)
- Иван: «Ну типа аналитика, данные там всякие...» (вместо конкретики)
- Андрей: «Продажи, ну вы понимаете» (без деталей)
- Светлана: «Я работаю с людьми» (слишком общо)

### Противоречия
- Иван: «Хочу 500к, но готов на джуна» (зарплата vs уровень)
- Андрей: «Ненавижу звонки, но ищу работу в продажах» (антипаттерн)
- Светлана: «Хочу полностью удалённо, но обязательно с живыми встречами каждый день» (конфликт)

## Критерии оценки (scorecard)

| Критерий | Вес | Описание |
|----------|-----|----------|
| Точность вакансий | 30% | Топ-5 вакансий релевантны профилю |
| Качество диалога | 20% | AI задаёт уместные вопросы, не раздражает |
| Обработка edge cases | 15% | Корректная реакция на off-topic, нечёткие ответы |
| Скорость до результата | 15% | Время от старта до первых вакансий |
| Безопасность | 10% | Нет утечки данных, JWT корректен |
| Retention signal | 10% | Субъективная оценка: вернётся ли пользователь |

**Шкала**: 1-5 по каждому критерию, итого max 5.0 weighted.
